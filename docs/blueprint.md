# **App Name**: Eternal Valentine

## Core Features:

- Romantic Hero: Displays a wallpaper-ready romantic hero section with a heartfelt message and a call to action using an image located at /public/wallpaper.jpg.
- Interactive Reasons: Presents a series of reasons why the user is loved, displayed one at a time in an interactive format. Navigated using a 'Next' button.
- Proposal Flow: A final page that presents the marriage proposal with 'YES' and 'NO' options.
- Animated Transitions: Smooth transitions between different sections and reasons, enhancing the user experience.
- AI enhanced sentimentality tool: Utilize AI to augment the included reasons with ones it generates from the web. The AI is responsible for making sure all AI-provided resons are factually and grammatically correct, and safe to use within the app.

## Style Guidelines:

- Primary color: Romantic red (#FF2E63), evoking love and passion.
- Background color: Desaturated deep red (#27070E) providing a luxurious backdrop.
- Accent color: Soft pink (#FF6B9D), highlighting interactive elements.
- Body and headline font: 'Playfair Display' (serif) for an elegant and romantic feel. Note: currently only Google Fonts are supported.
- Smooth, subtle animations for transitions and interactions, enhancing the romantic and engaging feel.
- Mobile-responsive layout ensuring the application looks elegant and functions well on all devices.
- Minimalist and elegant icons to enhance the UI without distracting from the core message.
